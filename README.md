## ğŸ’» Local Development

### Prerequisites

- Node.js >= 20.0
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/AbdulWahid126/AI-Humanoid-Robotics-Books.git
cd AI-Humanoid-Robotics-Books

# Install dependencies
npm install

# Start development server

```

The site will open at `http://localhost:3000`

### Build

```bash
# Create production build
npm run build

# Serve production build locally
npm run serve
```
